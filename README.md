## What is SECURELOOP?

SECURELOOP is a practical framework for Software Supply Chain Security. 
It guides you from SBOM generation to policy enforcement in 5 clear loops.

**Built for:** Banking • Defense • Public Sector • Critical Infrastructure  
**Compliance:** NIS2 • DORA • CRA • ISO27001

---

## The 5 Loops
```
DISCOVER → VERIFY → SECURE → ENFORCE → MONITOR
    ↑                                      │
    └──────────────────────────────────────┘
```

1. **DISCOVER** – What's in your software? (SBOM, Dependencies)
2. **VERIFY** – Is it safe? (CVE Scanning, License Check)
3. **SECURE** – Can you trust it? (Signing, Attestation)
4. **ENFORCE** – Stop what's unsafe (Policy-as-Code)
5. **MONITOR** – Stay secure over time (Continuous Scanning)
